Coupon System

  • Admin creates coupons — POST /api/payment/coupons
  • Scope: specific vendor or all vendors
  • Valid for: specific plan or all plans
  • Type: percentage discount or fixed amount
  • Expiry date + usage limit enforced
  • All uses tracked in coupon_usages table
